## 设计意图

购物车首版按截图实现商户分组、勾选状态、数量步进器、猜你喜欢和底部结算栏。

## 页面结构

- 页面根容器：`_pg`
- 页面容器：`PageShell`
- 页面运行时：`usePageRuntime`
- 页面状态订阅：`observer`
- 内容区域：商户分组商品、活动标签、猜你喜欢双列卡片。
- 固定底部：全选、金额汇总、结算/删除按钮。

## 动态与静态边界

- 接口图片：真实图片区域统一用项目封装 `AppImage`，render 内以空字符串变量预留地址，由组件承接加载中、淡入和失败态。
- 图标资源：优先使用项目封装；NutUI 有匹配项时先封装为项目组件，找不到匹配项时用图片组件预留空地址。
- 接口文本/数据：通过页面 service 获取。
- 代码渲染：页面结构、状态、交互和基础样式。
- 本地配置：页面标题、导航策略和分包注册。

## 状态要求

- loading：页面运行时统一承接。
- empty：优先使用 `BaseEmpty`。
- error：优先使用 `BaseException` 或 `StatusException`。
- 未登录：需要身份时使用 `usePageRuntime({ loginRequired: true })` 或 `AuthAction`。
- 降级态：可降级接口在 service 内返回默认值。

## 接口与 Service

| 模块 | service | 失败策略 | 是否阻断页面 |
|---|---|---|---|
| 页面数据 | `fetchCartData()` | service 内归一和兜底 | 按业务决定 |
| 加入购物车 | `addMallCartItem()` | 本地 storage 兼容读取，异常为空 | 否 |
| 当前购物车快照 | `replaceMallCartGroups()` | 商品编辑后保存当前分组，避免默认商品重新灌回 | 否 |

## 交互与跳转

- 右上角编辑可切换编辑模式，底部支持批量移入收藏和批量删除；单商品仍支持移入收藏和删除。
- 商品图片点击预览大图，商品标题点击进入商品详情。
- 商品勾选和数量修改实时更新底部金额；数量为 1 时继续点减号进入删除确认，取消则保留 1 件。
- 删除、移入收藏、改数量会写入当前购物车快照；再次进入页面应保持用户编辑后的购物车状态。
- 删除模式下无选择会提示；有选择时先弹微信确认 modal，再删除。
- 结算按钮会把当前勾选商品生成商城结算草稿 `draftId`，再进入确认订单。
- 猜你喜欢商品进入 `mall-product-detail`。

## 交互矩阵

| 元素 | 处理结果 |
|---|---|
| 编辑 / 完成 | 切换编辑模式 |
| 商品勾选 | 更新选中状态和金额 |
| 数量步进器 | 数量大于 1 时更新当前商品数量；数量为 1 时点减号触发删除确认 |
| 商品图片 | 微信预览商品大图 |
| 商品标题 | 进入商品详情 |
| 全选 | 切换全部商品选中 |
| 删除 | 微信确认后删除选中商品 |
| 批量移入收藏 | 写入本地收藏列表，从当前购物车移除并给出成功反馈 |
| 单商品移入收藏 | 写入本地收藏列表，从当前购物车移除并给出成功反馈 |
| 单商品删除 | 微信确认后删除当前商品 |
| 结算 | 选中商品后生成结算草稿并进入确认订单 |
| 猜你喜欢 | 进入商品详情 |

## 状态矩阵

| 状态 | 页面表现 |
|---|---|
| loading | `usePageRuntime` 初始化购物车 |
| 新增商品存在 | 置顶展示到当前购物车分组 |
| 商品删空 | 隐藏编辑入口和底部结算栏，展示统一空态，保留猜你喜欢继续承接 |
| 未选择商品 | 删除 / 结算给微信 toast 提示 |
| 删除确认取消 | 保持原购物车状态 |
| 数量为 1 点减号后取消删除 | 保持当前商品数量为 1 |

## 微信开发工具验收清单

- 从商品详情或商城首页加购后进入购物车，应看到新增商品。
- 编辑模式下删除商品，应先出现确认 modal。
- 未勾选点击结算或删除，应提示先选择商品。
- 编辑模式未勾选点击移入收藏，应提示先选择商品。
- 点击商品图应预览图片，点击商品标题应进入对应商品详情。
- 勾选多个商品后结算：确认订单应展示对应商品、数量、规格和赠品。
- 编辑模式点击移入收藏后进入我的收藏，应能看到移入的商品。
- 删除全部商品后，应展示统一空态，不应继续出现结算栏或编辑按钮。
- 删除静态默认商品后重新进入购物车，不应重新出现已删除商品。
- 商品数量为 1 时点击步进器减号，应弹出删除确认；取消后数量仍为 1，确认后移除该商品。
